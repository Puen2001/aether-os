# voice pack — python deps for capture.py, vad-capture.py, kokoro-synth.py.
# Install into the runtime venv (see README.md "Setup"):

kokoro-onnx     # local TTS engine (kokoro-synth.py)
numpy           # audio buffers (vad-capture.py)
onnxruntime     # Silero VAD inference (vad-capture.py; also an openwakeword dep)
openwakeword    # wake-word detection (capture.py, optional at runtime)
pynput          # global hotkey listener (capture.py)
sounddevice     # mic input streams (capture.py, vad-capture.py)
soundfile       # WAV writing (kokoro-synth.py)
